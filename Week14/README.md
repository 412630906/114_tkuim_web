# Steam Store Clone

這是一個模仿 [Steam](https://store.steampowered.com/) 商店介面的練習專案。
主要專注於還原其深色主題 (Dark Theme)、漸層配色與排版結構。

## 專案特色
- **視覺還原**：使用與 Steam 相同的色系 (`#171a21`, `#1b2838`) 與字體風格。
- **首頁輪播**：模擬首頁精選遊戲的輪播效果（左側大圖、右側資訊欄）。
- **遊戲列表**：包含價格標籤、折扣綠色標籤等細節。
- **RWD 響應式**：基本的響應式配置。

## 檔案結構
```
SteamClone/
├── index.html    # 主要入口
├── style.css     # 樣式表 (CSS Variables, Flexbox/Grid)
├── script.js     # 互動邏輯 (輪播圖切換)
├── todo.md       # 待辦清單
└── README.md     # 專案說明
```

## 如何預覽
1. 直接用瀏覽器開啟 `index.html`。
2. 或使用 VS Code 的 Live Server 套件開啟。
